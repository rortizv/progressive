Under the hood

This block was @defer-loaded — its code wasn't part of the page's initial JS bundle or SSR payload, only fetched once you clicked the button above.

This page renders with RenderMode.Server (fresh SSR on every request — that's why the API status above has a live timestamp). The /about page uses RenderMode.Prerender instead: rendered once at build time, then served as a static file forever.

GET /api/build-info is cached for 10s on the Nest side with a plain CacheInterceptor — ISR-style, same trade-off as Next.js's Incremental Static Regeneration. Click reload a few times fast: the value only changes once the 10s window has passed.
