Progressive

Angular SSR + NestJS, running in a single Node.js process.

@if (health.isLoading()) { Checking API health… } @else if (health.error()) { Could not reach /api/health } @else if (health.value(); as data) { API status: {{ data.status }} {{ data.timestamp }} }
  • Angular renders this page with real SSR (not just client-side)
  • NestJS serves /api/* and hosts the SSR engine above
  • Hydration reuses the server-rendered data — no double fetch

Next steps

  1. npm run build — builds the Angular app and the Nest server
  2. NG_ALLOWED_HOSTS=localhost npm start — runs the single process
  3. Edit examples/playground-server/src/app for API routes
  4. Edit examples/playground-web/src/app for the UI
